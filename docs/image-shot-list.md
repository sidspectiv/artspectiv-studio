# Image Shot List

Use this guide when selecting and naming photos for the Artspectiv Studio site.

## Folder

All images should live in:

```text
assets/
```

## Required image filenames

The current `index.html` expects these filenames. Replace the placeholder files with real compressed images using the same names.

```text
rita-hero.jpg
food-01.jpg
bar-01.jpg
interior-01.jpg
detail-01.jpg
people-01.jpg
rita-01.jpg
rita-02.jpg
rita-03.jpg
portfolio-food-01.jpg
portfolio-drinks-01.jpg
portfolio-interior-01.jpg
portfolio-food-02.jpg
portfolio-people-01.jpg
portfolio-drinks-02.jpg
portfolio-interior-02.jpg
```

## Selection rule

Use only the strongest 20–30 images across the whole site.

Good images should show at least one of these:

- Appetite: food looks immediately desirable.
- Atmosphere: the place feels worth visiting.
- Trust: staff, service, hands or process feel real.
- Texture: drinks, plates, interiors or details feel premium.
- Commercial use: the image could work on Instagram, Google or a booking platform.

## Compression rule

Before uploading:

- Long side: 1800–2400px
- Target file size: under 500KB per normal image
- Hero image: under 700KB if possible
- Format: JPG for photos, WebP if you want better performance later

## What not to include

- Repetitive angles
- Blurry low-light misses
- Over-edited food colours
- Too many personal/travel-style images
- Anything that does not help a local business owner imagine their own brand looking better
