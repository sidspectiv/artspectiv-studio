# Artspectiv commercial strategy

Use when work touches **outreach copy, pitch scripts, lead sell-sheets, GTM planning, or commercial positioning** — not routine `index.html` layout.

## Read order

1. **`CONTEXT.md`** — distilled pillars, package mapping, first-phase GTM pair.
2. **`docs/commercial-strategy.md`** — lead priority, corrected data points, scripts, capacity rules.
3. **Verify** lead stats (followers, reviews, awards) with a fresh check before drafting — report data goes stale.

## Guardrails

- **Do not change** site pricing floors (£445 / £945 / £1,595) or package names on `index.html` without explicit user request.
- **Do not quote** unverified metrics (e.g. Posh 2026 award, exact Treatwell review counts) in outreach or site copy.
- **Positioning:** “Monthly visual content…” — not vague “social media help”.
- **Outreach structure:** observation → gap → credibility → low-friction next step (not “I’m a local photographer” first).
